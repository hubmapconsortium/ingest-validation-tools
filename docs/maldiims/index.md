## Directory schema

| pattern | required? | description |
| --- | --- | --- |
| `csv/[^/]+\.csv` | ‚úì | Intensities M/Z values with Pixel location |
| `imzML/[^/]+\.ibd` | ‚úì | Index to imzML file |
| `imzML/[^/]+\.imzMl` | ‚úì | Mass spec raw data file |
| `metadata/[^/]+_LipidAssignments\.xlsx` | ‚úì | Microsoft Excel file containing the m/z, assignment, lipid class, etc. |
| `metadata/[^/]+_meta\.json` | ‚úì | JSON file containing the machine parameters/settings |
| `metadata/[^/]+_microscopy\.txt` | ‚úì | Transformations/map back to autofluorescence microscopy (related) data |
| `ometiffs/[^/]+_multilayer\.ome\.tiff` | ‚úì | Aligned multilayer OME TIFF file of the IMS data |
| `ometiffs/separate/[^/]+_mz[^/]+\.ome\.tiff` | ‚úì | Each file is a different M/Z value. |
| `extras/.*` |  | Free-form descriptive information supplied by the TMC |
| `extras/thumbnail\.(png\|jpg)` |  | Optional thumbnail image which may be shown in search interface |
